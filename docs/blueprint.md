# **App Name**: JavaCraft Portfolio

## Core Features:

- Hero Section: Visually striking introduction with headline, subheadline, bio, CTAs ('View Projects,' 'Contact Me'), and 'Download Resume' button.
- Skills Dashboard: Visually organized dashboard categorizing skills: Languages, Frameworks, Tools, Cloud. Allows quick addition/modification of skills.
- Experience Timeline: Vertical timeline component showing work history (Job Title, Company, Date, Achievements) which are easily editable.
- Featured Projects: Grid layout of project cards: Project Title, Description (problem solved), Tech Stack tags, GitHub/Live Demo links. Easy to add/remove projects.
- Education & Certifications: A dedicated place to present credentials. List degrees and certifications with titles and brief details.
- Contact Section: Includes a styled contact form and a 'Send Email' API tool that can send direct email when the user clicks.
- Navigation & Routing: Sticky header with smooth scrolling to sections. Fully responsive layout adaptable to various screen sizes. Maintain navigation states using tool with URL parameter synchronization.

## Style Guidelines:

- Primary color: Navy blue (#1A237E) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5), providing a clean and modern backdrop.
- Accent color: Teal (#008080) to highlight key elements such as call-to-action buttons and links, ensuring they stand out against the primary and background colors.
- Font pairing: 'Inter' (sans-serif) for body and headlines, offering a modern, neutral, and highly readable feel.
- Use Lucide-React icons throughout the portfolio for a consistent and professional look.
- Utilize a responsive grid layout for project cards and skill dashboards, ensuring a seamless experience on all devices.
- Subtle hover animations on interactive elements like buttons and links to provide clear feedback and enhance user engagement.
